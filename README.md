# Women Menstruation Aid

An empathetic AI-powered web application designed to help women manage and relieve menstrual pain.

## Core Features
- Live Relief Coach: Real-time guided sessions with adaptive breathing, heat therapy, and stretches.
- Smart Nutrition Remedy Builder: Personalized anti-inflammatory recipes and meal suggestions.
- Symptom tracking and predictions.
- AI chatbot for instant support.

## Getting Started

1. Clone the repository
2. Set up backend and frontend separately (see their READMEs)
3. Run with docker-compose up for full stack

## Tech Stack
- Backend: Python + FastAPI
- Frontend: JavaScript + React
- AI: LLM integration (Groq, OpenAI, etc.)
